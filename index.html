A simple house phone for kids

This project lets you create a normal house phone for your kids using a regular phone, a small adapter box, and a cheap internet phone provider.

It feels like a traditional landline, but instead of using the telephone company it uses your home internet.

Old phone ↓ OBi200 adapter ↓ Home router / internet ↓ VoIP.ms ↓ Mom, Dad, Grandma

Why parents like this setup

  • No smartphone required
  • Kids can call family easily
  • Very cheap compared to a landline
  • Works with any old phone
  • Feels like a normal house phone

What you need

  • Old house phone (corded or cordless)
  • Polycom OBi200 VoIP adapter
  • Internet router
  • VoIP.ms account

How the system works

Phone ↓ OBi200 ↓ Router ↓ VoIP.ms ↓ Other phone

The OBi200 acts like the phone company. It powers the phone and converts the call into internet data.

Step-by-step setup

1. Create a VoIP.ms account
Sign up at voip.ms and add prepaid credit. Buy a phone number.
2. Connect the hardware
Phone → OBi200 → Router
Plug the phone cable into the OBi200 phone port, connect ethernet to your router, and plug in power.
3. Find the adapter IP address
Pick up the phone connected to the OBi200 and dial:
***1
The device will read an IP address out loud.
4. Open the settings page

Type the IP address into a browser.

username: admin password: admin
5. Enter VoIP.ms settings

Enter:

  • SIP server
  • username
  • password
  • phone number

Save the settings.

6. Test the phone

Call your cell phone. Then call the house phone number to make sure it rings.

Make it kid-friendly

Add speed dial buttons so kids can easily call family.

1 → Mom 2 → Dad 3 → Grandma 4 → Grandpa

You can also tape a small contact card next to the phone.

Frequently asked questions

Can kids call anyone?

Yes by default. You can limit dialing or only use speed dial buttons.

Who powers the phone?

The OBi200 supplies the phone-line power just like a traditional landline.

Is this difficult to set up?

Most people can set it up in about 20-30 minutes.
